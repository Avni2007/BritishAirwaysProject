✈️ AI-Driven Predictive Customer Satisfaction and Personalized Experience System
📌 Project Overview

This project focuses on analyzing airline customer reviews to predict customer satisfaction and provide personalized recommendations using Machine Learning and Natural Language Processing (NLP). It helps airlines understand passenger sentiment and improve services proactively.

🎯 Objectives
Analyze customer reviews using sentiment analysis
Predict customer satisfaction level (Happy / Neutral / Unhappy)
Identify key factors affecting customer experience
Provide personalized improvement suggestions
Build a machine learning classification model
🧠 Technologies Used
Python 🐍
Pandas (Data Handling)
NumPy
Scikit-learn (Machine Learning)
VADER Sentiment Analysis (NLP)
Matplotlib / Seaborn (Data Visualization)
📂 Dataset
Airline customer review dataset (CSV file)
Contains fields like:
Review text
Ratings
Customer feedback
Other metadata (if available)
⚙️ Workflow
Data Collection
Load airline review dataset
Data Preprocessing
Remove null values
Clean text data
Drop unnecessary columns
Sentiment Analysis
Use VADER sentiment analyzer
Convert reviews into sentiment scores
Model Training
Split dataset into train/test sets
Train Random Forest Classifier
Prediction
Predict customer satisfaction category
Evaluation
Accuracy score
Confusion matrix
📊 Model Used
Random Forest Classifier
NLP-based sentiment scoring (VADER)
🚀 How to Run the Project
1. Install dependencies
pip install pandas numpy scikit-learn vaderSentiment matplotlib seaborn
2. Run the Python script
python main.py
